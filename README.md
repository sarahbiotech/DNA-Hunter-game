# DNA Nucleotide Hunter

DNA Nucleotide Hunter:  an educational game designed for biology learners and beginner programmers. The goal is to shoot complementary DNA bases: Adenine (A) pairs with Thymine (T), and Cytosine (C) pairs with Guanine (G). Players practice base pairing while having fun!

## Features
- Interactive gameplay to reinforce DNA base pairing knowledge
- Score tracking
- Reset game button
- Works in web browsers (HTML/JavaScript version)
- Python version available for desktop

## Versions
1. Web version (HTML/JavaScript) – Playable in any modern browser.
2. Python version – Run locally using Python 3.x and Pygame.

## How to Play
- For the web version, open `dnagame.html` in a browser.
- For the Python version, run `game.py` in your Python environment.

## Requirements
- Web version: Modern browser (Chrome, Firefox, Edge, Safari)
- Python version: Python 3.x, Pygame library
